
1. api    网络请求接口的定义


2.  app    application


3.  base   基类 (V层所有的基类activity和fragment，M层的基类，P层的基类）
        V层(a.初始化V层的界面。b.初始化P的接口调用。c.初始化当前界面的数据)


4.  interfaces   mvp业务逻辑的接口定义（各个业务的M,V,P层的接口的定义---契约类）



5.  module    网络请求M层的实现类和对应的数据bean类(实现各个业务的M层接口的功能)


6.  net       网络数据加载的类


7.   persenter     p层所有业务的实现类(实现各个业务的P层接口的功能)
                    p层实现类里面关联V层的接口，创建对应的M层并且与之关联


8.  ui    v层所有业务的实现类(实现各个业务V层的接口功能)
            a.提供当前V层所对应的P层的实现类。 b.实现对应的V层的接口，并处理相对应的数据


9. utils    工具类
